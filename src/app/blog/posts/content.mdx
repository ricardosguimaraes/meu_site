## Avatar

Substitua o arquivo `public/images/avatar.jpg` pela sua própria imagem de avatar. Ela é usada na página `/about` e também nas imagens dinâmicas de open-graph.

## Favicon

Substitua o arquivo `src/app/favicon.ico` pelo seu próprio favicon.

## Conteúdo

Substitua o conteúdo do arquivo `resources/content.js` pelo seu próprio conteúdo. Você pode usar componentes personalizados na maioria dos casos, já que as props geralmente são declaradas como `ReactNode` ao invés de `string`, mas é importante importar esses componentes no arquivo.

Os imports acima permitem usar os componentes `InlineCode` e `Link` no arquivo de conteúdo.

### Dados pessoais

Seus dados pessoais são usados em todo o app para renderizar títulos, labels e imagens personalizadas. Os idiomas são exibidos na página `/about`, e localização e horário no cabeçalho.

<CodeBlock
  marginBottom="16"
  highlight="2-3,7-10"
  codes={[
    {
      code: `const person = {
    firstName: "Selene",
    lastName: "Yu",
    get name() {
      return "...";
    },
    role: "Design Engineer",
    avatar: "/images/avatar.jpg",
    location: "Asia/Jakarta", // Espera-se o identificador de fuso horário IANA, ex: 'Europe/Vienna'
    languages: ["English", "Bahasa"], // opcional: Deixe o array vazio se não quiser exibir idiomas
};`,
      language: "tsx",
      label: "src/app/resources/content.js",
    },
  ]}
/>

### Links sociais

Os links sociais são exibidos na página `/about` e no rodapé, baseados no array configurado no objeto `social`. Você pode definir ícones personalizados para cada um, mas não esqueça de importá-los em `src/once-ui/icons.ts`. Leia a [documentação do Once UI](https://once-ui.com/docs/icons) para mais informações.

<CodeBlock

];`,

    },

/>

### Páginas

Cada página tem seu próprio objeto que gerencia o conteúdo personalizado, como títulos, subtítulos, imagens e dados dinâmicos.

<CodeBlock
  marginBottom="16"
  codes={[
    {
      code: `const home = {
    label: "Início",
    title: "Portfólio de Selene Yu",
    description: "Site de portfólio apresentando meu trabalho como Design Engineer",
    headline: <>Engenheira de design e criadora</>,
    subline: (
      <>
        Sou Selene, engenheira de design na <InlineCode>FLY</InlineCode>, onde crio experiências de usuário intuitivas.
        <br /> No tempo livre, desenvolvo meus próprios projetos.
      </>
    ),
};`,
      language: "tsx",
      label: "src/app/resources/content.js",
    },
  ]}
/>

A propriedade `label` é usada na navegação do cabeçalho. A propriedade `title` é usada no título da página e a propriedade `description` é usada na descrição da página. Elas também são usadas em meta tags e open graph.
