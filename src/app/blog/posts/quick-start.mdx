## Sobre

Magic Portfolio é um template de portfólio completo, baseado em MDX, SEO-friendly e responsivo, construído com Once UI e Next.js.

## Licença

Magic Portfolio é licenciado sob a [CC BY-NC 4.0](https://creativecommons.org/licenses/by-nc/4.0/). Você só pode usá-lo para fins pessoais e deve atribuir o trabalho original. A atribuição é adicionada no rodapé por padrão, mas você pode colocá-la em qualquer outra parte visível do seu site.

Assine o [Once UI Pro plan](https://once-ui.com/pricing) para estender a licença para [Dopler CC](https://once-ui.com/products/magic-portfolio).
