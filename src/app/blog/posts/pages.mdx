## Ativar ou desativar páginas

O componente `RouteGuard` do Magic Portfolio cuida de renderizar páginas condicionalmente com base no objeto `routes` do arquivo `resources/config.js`.

<CodeBlock
  marginBottom="16"
  highlight="6"
  codes={[
    {
      code: `const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
};`,
      language: "tsx",
      label: "src/app/resources/config.js",
    },
  ]}
/>

O código acima garante que a página `/gallery` não estará acessível e não será exibida na navegação.

## Adicionar novas páginas

Ao criar uma nova página, ela precisa ser adicionada ao objeto `routes` no arquivo `resources/config.js`.

<CodeBlock
  marginBottom="16"
  highlight="6"
  codes={[
    {
      code: `const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
    '/music': true,
};`,
      language: "tsx",
      label: "src/app/resources/config.js",
    },
  ]}
/>

O código acima garante que a página `/music` estará acessível. Os usuários poderão navegar até ela após adicioná-la ao menu de navegação no arquivo `components/Header.tsx`.
