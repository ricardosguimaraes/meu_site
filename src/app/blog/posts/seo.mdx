## Meta tags

O Magic Portfolio gera automaticamente meta tags e open graph para suas páginas com base no arquivo `content.js`.

## Schema

O Magic Portfolio gera automaticamente tags de schema para suas páginas com base no arquivo `content.js`.

## Imagem Open Graph

Imagens para compartilhamento social (open-graph e twitter) são geradas automaticamente com `next/og`.
