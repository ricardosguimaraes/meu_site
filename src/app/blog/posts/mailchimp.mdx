## Conta Mailchimp

Crie uma [conta Mailchimp](https://mailchimp.com) e crie uma nova lista caso ainda não tenha uma.

## Formulário incorporado

Crie um novo [formulário incorporado](https://mailchimp.com/help/add-a-signup-form-to-your-website/) e copie a URL do formulário para a configuração do seu Magic Portfolio.

<CodeBlock
  highlight="2"
  codes={[
    {
      code: `const mailchimp = {
    action: 'https://url/subscribe/post?parameters',
    effects: {
      ...
    }
};`,
      language: "tsx",
      label: "src/app/resources/config.js",
    },
  ]}
/>

## Conteúdo da newsletter

Você pode modificar o título e a descrição no arquivo `content.js`.

<CodeBlock
  highlight="2"
  codes={[
    {
      code: `const newsletter = {
    display: true,
    title: <>Inscreva-se na Newsletter de {person.firstName}</>,
    description: (
      <>
        Ocasionalmente escrevo sobre design, tecnologia e compartilho pensamentos sobre a interseção entre criatividade e engenharia.
      </>
    ),
};`,
      language: "tsx",
      label: "src/app/resources/content.js",
    },
  ]}
/>

## Efeito de fundo

Há um fundo pré-configurado em `Mailchimp.tsx` que você pode modificar no objeto `mailchimp`. Defina elementos gráficos como gradiente, pontos, linhas e grid e configure sua aparência para o bloco de newsletter.

<CodeBlock
  codeHeight={24}
  marginBottom="16"
  codes={[
    {
      code: `const mailchimp = {
    action: 'https://url/subscribe/post?parameters',
    effects: {
      mask: {
        cursor: false,
        x: 50,
        y: 0,
        radius: 100
      },
      gradient: {
        display: true,
        x: 50,
        y: -25,
        width: 100,
        height: 100,
        tilt: 0,
        colorStart: 'accent-background-strong',
        colorEnd: 'static-transparent',
        opacity: 50
      },
      dots: {
        display: true,
        size: 2,
        color: 'brand-on-background-weak',
        opacity: 20
      },
      lines: {
        display: false,
        color: 'neutral-alpha-weak',
        opacity: 100
      },
      grid: {
        display: false,
        color: 'neutral-alpha-weak',
        opacity: 100,
        width: 'var(--static-space-32)',
        height: 'var(--static-space-32)'
      }
  }
}`,
      language: "tsx",
      label: "src/app/resources/config.js",
    },
  ]}
/>
